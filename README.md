# Excel 自动化助手

一个面向日常办公、运营和财务场景的表格处理小工具，用来减少重复整理 Excel / CSV 的时间。

## 这个工具解决什么问题
- 多个表格格式不一致，手工整理很耗时
- 名单重复、字段混乱、数据不好合并
- 对账、差异检查、汇总报表经常重复做
- 每周、每月都要做同样的数据清洗和整理

## 核心功能方向
- 表格去重、合并、拆分
- 列名统一、格式清洗、空值处理
- 对账与差异检查
- 固定格式报表生成
- CSV / Excel 批量整理
